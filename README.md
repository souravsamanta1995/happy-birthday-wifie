# 🎉 Barnali's Birthday Website

A beautiful, animated birthday celebration website created with love for **Barnali's 24th Birthday** (May 16, 2000).

From: **Sourav** 💕

---

## ⚡ Quick Start

1. **Add 28 photos** to `/public/images/` folder (exact filenames required)
2. **Run the website**: `npm run dev`
3. **Share with Barnali!** 🎊

✨ **Animated Hero Section**

- Personalized birthday greeting
- Floating blob animations
- Birthday info display
- Bouncing celebration emoji

🎉 **Celebration Effects**

- 40 falling emoji particles

- Festive atmosphere
